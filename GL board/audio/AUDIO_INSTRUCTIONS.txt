ANNOUNCEMENT AUDIO FILES (works in any kiosk browser)

Put all these MP3 files in THIS folder (public/audio/).
Make them at a free TTS site (ttsmaker.com, naturalreaders.com): type the
letter/number/sentence, choose the language voice, download MP3, rename.

NAMING MUST BE EXACT: all lowercase, .mp3 extension.

PLAY ORDER when a car is Ready:
  chime
  -> plate in ENGLISH  + msg-en
  -> plate in MALAY    + msg-ms   (letters english, digits ms-#)

Missing files are skipped automatically (no error), so you can add
languages in stages. After adding files: npm run build then firebase deploy.
